# 成果展示準備工作流程 (Presentation Prep Workflow)

## 目的

本工作流程旨在協助黑客松參賽團隊準備具說服力的簡報、演示稿和技術文件，確保在展示時能充分傳達解決方案的價值和影響力。

## 如何調用

這個工作流程是 `standalone: true`，因此可以直接由使用者調用。

在 BMAD 環境中，您可以透過以下指令啟動它：

```bash
workflow /bmad/mhp/workflows/presentation-prep/
```

## 預期輸入

本工作流程為中互動性，將在關鍵決策點引導使用者提供輸入。使用者需要準備好：

*   已完成的解決方案概念或原型。
*   對目標受眾和展示目標的初步想法。
*   團隊成員的溝通和演示專長。

## 生成輸出

本工作流程將在執行過程中，透過對話引導團隊完成展示準備。最終會生成一份摘要報告，記錄展示準備的關鍵決策、內容大綱和演練回饋：

*   **預設輸出檔案**：`{output_folder}/presentation-prep-report-{{date}}.md`

## 特殊要求

*   本工作流程為中互動性，需要使用者在關鍵決策點提供確認和輸入。
*   建議在團隊已完成解決方案開發後使用。
